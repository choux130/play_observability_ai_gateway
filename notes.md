
### Local Setup
1. Install terraform 
```sh
brew install hashicorp/tap/terraform
```
2-1. Install Azure CLI
```sh
brew update && brew install azure-cli
```
2-2. Install AWs CLI
```sh
brew update && brew install awscli
```
3. Connect to Azure
```sh
az login 
az account list
```
4. Run terraform 
```sh
terraform init
terraform plan
terraform apply
```
